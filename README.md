# GitLab-TreeView

Chrome extension to show a code tree on GitLab  :)

## Features

- Quick preview code
- GitLab UI style
- Support gitlab 9.x version

## Installation

[Chrome](https://chrome.google.com/webstore/detail/gitlab-treeview/kfjchffabpogdehadpflljaikjicdpng)
